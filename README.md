# 🎮 Tic Tac Toe

A modern and interactive Tic Tac Toe game built with React, featuring smooth gameplay, responsive UI, and engaging sound effects. The project includes local multiplayer and a foundation for future online gameplay.

---

## 🚀 Features

- 🧑‍🤝‍🧑 Local Multiplayer (2 players on same device)
- 🔊 Sound effects for moves and winning
- ⚡ Fast and responsive UI
- 🧠 Accurate win detection logic
- 🔄 Restart game anytime
- 🧭 Simple menu navigation
- 🌐 Online mode UI (coming soon)

---

## 🛠️ Tech Stack

- React
- JavaScript
- CSS / Tailwind
